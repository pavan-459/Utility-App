# Utility Convertor

A clean, fast, everyday converter app — no installs, no login, works offline after first load.

**Live:** <https://pavan-459.github.io/Utility-App/>

## Features

| Converter | Units |
| --- | --- |
| Currency | 30+ currencies with live rates (USD → INR and more) |
| Length | mm, cm, m, km, in, ft, yd, mi, nautical mile |
| Weight | mcg, mg, g, kg, metric ton, oz, lb, stone |
| Temperature | Celsius, Fahrenheit, Kelvin |

- Live currency rates via [Frankfurter](https://www.frankfurter.app/) (free, no API key)
- Quick-pair buttons for common conversions (USD → INR, EUR → INR, etc.)
- Dark / light mode with system preference detection
- Mobile responsive

## Usage

Open `index.html` directly in any browser — no build step, no server needed.

## Hosting

Deployed on GitHub Pages. To deploy your own fork:

1. Go to **Settings → Pages**
2. Source: `main` branch, `/ (root)`
3. Save — live at `https://<username>.github.io/<repo>/`
